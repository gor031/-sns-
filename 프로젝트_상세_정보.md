# 🎨 카드뉴스 스튜디오 프로젝트 상세 명세서

이 문서는 현재 진행 중인 **카드뉴스 생성기 및 디자인 스튜디오** 프로젝트의 구조, 주요 기능, 그리고 핵심 로직을 정리한 문서입니다.

---

## 🚀 프로젝트 개요

본 프로젝트는 사용자가 간단한 키워드나 주제를 입력하면 AI(Gemini/GPT)를 통해 카드뉴스 원고를 생성하고, 이를 세련된 디자인의 카드뉴스로 변환하거나 전문가급 '디자인 스튜디오'에서 직접 편집할 수 있는 웹 어플리케이션입니다.

---

## 🛠 기술 스택

- **Frontend**: React (TypeScript), Vite
- **Styling**: Tailwind CSS
- **Graphics**: Fabric.js (디자인 스튜디오 핵심 라이브러리)
- **Icons**: Lucide React
- **Export**: html2canvas, JSZip, file-saver
- **AI Integration**: Gemini Service (프롬프트 엔지니어링 기반)

---

## 🌟 주요 기능

### 1. AI 카드뉴스 에디터 (`App.tsx`)
- **3단계 프로세스**:
    1. **SNS 제목/내용 생성**: 소셜 미디어용 홍보 문구 생성.
    2. **원고 기획**: 슬라이드별 핵심 문구(최대 2문장) 및 강조 단어 기획.
    3. **JSON 변환**: 앱에서 즉시 렌더링 가능한 JSON 데이터 구조 생성.
- **실시간 미리보기**: 선택한 테마와 배경 이미지가 적용된 카드뉴스를 즉시 확인.
- **테마 시스템**: 70개 이상의 감각적인 컬러 팔레트 및 디자인 테마 제공.

### 2. 프리미엄 디자인 스튜디오 (`DesignStudio.tsx`)
- **캔버스 기반 편집**: Fabric.js를 활용한 자유로운 요소(텍스트, 도형, 이미지) 배치 및 수정.
- **레이어 관리**: 요소별 순서 변경, 잠금, 가시성 조절.
- **속성 패널**: 폰트, 크기, 정렬, 자간/행간, 그림자, 투명도 등 정밀 제어.
- **AI 매직 툴**: 
    - **배경 제거 (누끼)**: 이미지에서 피사체만 남기고 배경을 투명하게 처리.
    - **개체 분리 & 레이어 분해**: 단일 이미지를 AI가 분석하여 텍스트, 도형, 사물 레이어로 자동 분해.

### 3. 템플릿 마켓플레이스 (`Marketplace.tsx`)
- **사전 제작 템플릿**: 전문가가 디자인한 다양한 카테고리의 카드뉴스 템플릿 제공.
- **즉시 적용**: 마켓에서 선택한 템플릿을 에디터나 스튜디오로 즉시 불러와 편집 가능.

## 📋 핵심 데이터 구조 (JSON)

AI가 생성하고 앱이 소비하는 데이터의 표준 규격입니다.

```json
{
  "topic": "주제",
  "targetAudience": "타겟 독자",
  "tone": "어조",
  "hashtags": ["태그1", "태그2"],
  "slides": [
    {
      "pageNumber": 1,
      "header": "강렬한 **헤드라인**",
      "body": "본문 내용(줄바꿈 포함)",
      "headerStyle": { "align": "center", "fontSize": "text-5xl" }
    }
  ],
  "themeIndex": 0,
  "customBackgroundImage": "URL"
}
```

## 🔧 주요 컴포넌트 구조

- `App.tsx`: 메인 진입점 및 탭 네비게이션, AI 에디터 로직.
- `components/CardPreview.tsx`: 카드뉴스 미리보기 및 테마 렌더링 엔진.
- `components/studio/DesignStudio.tsx`: Fabric.js 기반의 고성능 디자인 편집기.
- `components/studio/Toolbar.tsx`: 스튜디오 도구 모음 (도형 추가, 업로드 등).
- `components/studio/PropertyPanel.tsx`: 선택된 요소의 상세 속성 제어 패널.
- `services/geminiService.ts`: AI 응답 파싱 및 데이터 정제 서비스.
